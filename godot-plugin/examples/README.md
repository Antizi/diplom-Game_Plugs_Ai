# Примеры интеграции (не часть аддона)

Скопируйте нужные фрагменты в **свой** проект Godot.

| Файл | Назначение |
|------|------------|
| `game_manager.gd` | Шаблон autoload или менеджера: `initialize`, `start_new_game`, `end_game`, `track` |
| `test_integration.gd` | Минимальная проверка cloud после настройки URL в редакторе |

Полный контракт API: [docs/integration.md](../../docs/integration.md).
