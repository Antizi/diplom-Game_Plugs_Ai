# LSTM-модель для адаптации игрового геймплея

Проект курсовой работы по теме: **"Адаптация игрового геймплея на основе анализа последовательностей действий игрока с использованием LSTM-нейронных сетей"**

## Описание

LSTM-модель для классификации стиля игры игрока на основе последовательности его действий. Модель анализирует ключевые слова действий игрока и определяет его стиль игры (исследовательский, скрытный или боевой), что позволяет адаптировать геймплей под предпочтения игрока.

## Архитектура модели

- **Embedding Layer** (32 измерения) - преобразование слов в векторные представления
- **LSTM Layer** (64 скрытых единиц) - обработка последовательностей переменной длины
- **Mean Pooling** - извлечение признаков (в прототипе)
- **Классификатор** (Linear → ReLU → Linear) - классификация на 3 класса

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
python lstm_prototype.py
```

Программа выполнит:
1. Обучение модели на датасете
2. Создание графиков процесса обучения (training_history.png)
3. Вывод примеров предсказаний на тестовом наборе

## Расположение в репозитории

Инференс в проде: `ml/main.py` → `POST /predict` (вызывается из backend).
